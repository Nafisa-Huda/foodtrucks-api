Foodtruck for every US state API

A simple API for that returns the top foodtruck in each US state.

Documentation
Try It Out

Please use for the full JSON object provided by this API: https://foodtruck-by-state-api.herokuapp.com/states

How to access individual foodtrucks according to the specified state:

For the direct link to the individual object (JSON): https://foodtruck-by-state-api.herokuapp.com/states/stateName

For the JS formatted fetch parameter: `https://foodtruck-by-state-api.herokuapp.com/states/${stateName}`

Important Notes:

The direct link to the individual stateName object will only need the "stateName" query parameter changed to the specific name of the US state being requested. The stateName query parameter can be substituted by any of the US states. It is spelling specific with spaces separating each word and is NOT case-sensitive.

For the JS formatted fetch parameter, make sure to include backticks (`) in your code to use the template literal in your JavaScript.

There are 2 keys that come with each "states" object:

name: name of state

cuisine: cuisine of the food
