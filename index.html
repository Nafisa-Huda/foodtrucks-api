Foodtruck for every US state API

Please use: https://foodtruck-by-state-api.herokuapp.com/

Documentation

How to access individual foodtrucks according to state:

For the direct link to the individual object (JSON): https://foodtruck-by-state-api.herokuapp.com/

For the JS formatted fetch parameter: `https://foodtruck-by-state-api.herokuapp.com/api/${state}`
