Foodtruck for every US state API

Please use for the full JSON object provided by this API: https://foodtruck-by-state-api.herokuapp.com/states

Documentation

How to access individual foodtrucks according to state:

For the direct link to the individual object (JSON): https://foodtruck-by-state-api.herokuapp.com/states/stateName

For the JS formatted fetch parameter: `https://foodtruck-by-state-api.herokuapp.com/states/${stateName}`

The direct link to the individual stateName object will only need the "stateName" query parameter changed to the specific state being requested. The stateName query parameter can be substituted by any of the US states. It is spelling specific with spaces separating each word and is NOT case-sensitive.

There are 2 keys that come with each US state. One being the name of the foodtruck and the second being the cuisine of the food.
